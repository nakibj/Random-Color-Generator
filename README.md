# Random Color Generator

A simple, interactive color generator that creates random hex colors and automatically copies them to your clipboard with a single click.

## Features

- **Random Color Generation** - Generates a new random hex color on every click
- **Background Color Change** - Entire background updates to the generated color
- **Hex Code Display** - Shows the hex code of the current color
- **Auto-Copy to Clipboard** - Click the hex code to automatically copy it to clipboard

## How It Works

1. Click anywhere on the screen (or click the hex code)
2. A random hex color is generated
3. Background changes to that color
4. Hex code is displayed on screen
5. Hex code is automatically copied to your clipboard
6. Paste the color code anywhere you need it
